# boiler
General boilerplate I use across various projects.

## Files

#### `.npmignore`

Used when publishing to NPM.

#### `build.gradle`

Gradle build file for building Java projects.

#### `jitpack.yml`

Configuration for [Jitpack](https://jitpack.io/).

#### `tailwind.config.js`

[Tailwind CSS](https://tailwindcss.com/) configuration.

#### `tailwind.css`

Base CSS file for Tailwind CSS.

#### `tsconfig.json`

TypeScript configuration.
